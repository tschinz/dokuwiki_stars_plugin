## Syntax
Stars can be created with two methods:
  - With the Toolbarbutton Stars 1-5
  - With Syntax code as shown below

```
{{stars>num}}
{{stars>-1/5}}
{{stars>-5/5}}
{{stars>}}
{{stars>1/5}}
{{stars>2/5}}
{{stars>3/5}}
{{stars>4/5}}
{{stars>5/5}}
{{stars>5.5/7}}
{{stars>700/1000}}
```
